# スネークゲーム - ソースコード

リンゴをゲットするゲーム。
10個ゲットすると、ゲットしたリンゴを-1する「どくろのリンゴ」と+5する「かご入りのリンゴ」が出現するようになる。
画面端に触れるとゲームオーバーとなり、最後にゲットしたリンゴの数が表示される。
